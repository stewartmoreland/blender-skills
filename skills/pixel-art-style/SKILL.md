# Pixel Art Style

Texel discipline. Crisp pixels. Limited palette.

## Tenets
1. Declare target texel size / texture res
2. Nearest filtering intent
3. Limited palette
4. Avoid thin triangles that shimmer
5. Ortho or locked angles help readability
