# Noir Style

Shadows as design. Silhouettes. Wet ground mirrors. Limited color or mono.

## Tenets
1. Motivated hard keys / blinds / neon slash
2. Negative fill
3. Smoke/fog shafts
4. Costume readable in silhouette
5. Grade crushed but detailed shadows
