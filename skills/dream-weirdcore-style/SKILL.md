# Dream / Weirdcore Style

Nostalgic spaces. Soft dread. Surreal edits. Childhood UI/objects out of place.

## Tenets
1. Soft fog / bloom
2. Childhood objects oversized or abandoned
3. Text overlays / cryptic captions optional (comp)
4. Unclear geography
5. Emotion > logic
